# 명당노트 리포트 설계 전략서

> **목표**: 사용자가 리포트를 보는 순간 "이걸 안 보면 손해"라고 느끼고, 유료 결제까지 자연스럽게 이어지는 리포트 구조 설계
>
> **근거**: NotebookLM 37개 소스 (전환 심리학, 페이월 벤치마크, UX 리서치, 프라이싱 전략) 종합

---

## 핵심 원칙: "보여주되, 완성하지 않는다"

무료→유료 전환의 핵심 메커니즘은 **"가치의 존재를 인지시키되 접근을 제한"**하는 것이다.

- 자이가르닉 효과: 미완료 과제가 완료 과제보다 **1.9배** 더 강하게 기억된다
- 손실 회피: 손실이 동일한 크기의 이득보다 **약 2배** 강하게 체감된다
- 블러 미리보기: 틴더의 블러 전략 → 사용자 **8%가 업그레이드** (연 $12억 매출)

**따라서 Free는 '방향'만, Paid는 '근거 숫자'를 판다.**

## 1. 리포트 정보 계층 구조

### 분석의 3단계와 요금제 매핑

| 분석 깊이 | 성격 | 티어 | 역할 |
|-----------|------|------|------|
| **설명적 분석** (Descriptive) | "지금 어떤 상태인가?" | Free | 호기심 자극, 정보 격차 생성 |
| **예측적 분석** (Predictive) | "앞으로 어떻게 될까?" | Paid | 불확실성 해소, 리스크 헷지 |
| **처방적 분석** (Prescriptive) | "그래서 뭘 해야 하나?" | Paid | 행동 지침, 의사결정 완결 |

> **규칙**: 무료는 "우리 동네 상권이 뜨고 있다"를 알려주지만, "그래서 내가 지금 이 가게를 계약해야 하는가?"에는 **침묵**한다.

## 2. Free Tier — "감질나는" 3대 지표

Free Tier의 목표는 **정답 제공이 아니라, 사용자 머릿속에 '지식의 구멍(Curiosity Gap)'을 만드는 것**이다.

### 지표 1: 상권 온도 스코어 (0~100)

**무료로 보여주는 것:**
- 종합 점수 + 추세(↑/→/↓) + 백분위(상위 N%)
- "이 골목은 뜨거워지는 중 🌡 78점 (상위 15%)"

**잠그는 것 (결제 트리거):**
- "왜 78점인가?" → 유동/매출/경쟁/임대 **기여도 분해**
- 최근 12개월 분해 추세 차트

**심리 작동**: 점수는 봤는데 **왜 그런지** 궁금해진다. "내가 모르는 설명 변수를 알면 불확실성이 줄어든다"는 상태가 탐색을 강화한다.

### 지표 2: 피크타임 Top-3

**무료로 보여주는 것:**
- 가장 붐비는 시간대 **3개만** 하이라이트
- "평일 12~13시 ⭐ / 금 18~20시 / 토 15~17시"

**잠그는 것:**
- 24×7 전체 히트맵 (블러 처리)
- 성별/연령 필터
- 방문 인구 유형(거주/직장/방문) 구분

**심리 작동**: Top-3를 보면 즉시 "그럼 나머지 시간은? 요일별 차이는? 20대 여성 비중은?" 추가 질문이 생긴다. 무료는 **"가능성"**을 보여주고, 유료는 **"운영 설계도"**를 판다.

### 지표 3: 경쟁 포화도 레이더

**무료로 보여주는 것:**
- 반경 내 동종업종 **점포 수 + 백분위 + 지도 점 표시**
- "반경 500m 내 카페 12개 (밀집도 상위 30%)"

**잠그는 것:**
- 경쟁의 **질** (매출 상위/하위 분포)
- 신규/폐업 흐름 (카니발라이제이션)
- 수요-공급 갭 분석

**심리 작동**: 경쟁이 많으면 **공포(손실회피)**가, 적으면 **기대(기회)**가 생긴다. 이 감정은 "지금 더 확인하지 않으면 손해 볼 수 있다"로 연결된다. **"개수만 주고, 강도는 유료"**가 전환을 만든다.

### 보너스 Free: 예상 창업 비용 계산기

- 보증금/권리금/인테리어를 입력하면 **총 투입 비용** 산출
- "이 자리에 카페를 차리려면 최소 **1억 8천만 원** 투입"
- 억 단위 숫자를 보는 순간 → 자본 상실 **공포** 극대화 → "몇만 원 내고 검증하는 게 합리적" → **유료 결제 당위성 자가 설득**

## 3. 페이월 설계 — 스크롤 35~45% 지점

### 최적의 타이밍

| 트리거 | 스크롤 위치 | 심리 레버 |
|--------|----------|---------|
| **경쟁사 매출 차트 직전** (1순위) | 35~45% | 호기심 + 손실회피 + 소유효과 |
| 생존확률/리스크 직전 | 50~60% | 공포 기반 손실회피 |
| 기회 지도/액션 직전 | 65~80% | 목표 달성 + FOMO |

**왜 35~45%가 최적인가:**
1. 사용자가 무료 요약을 충분히 본 후 → 신뢰 확보
2. 업종/주소 입력이라는 개인화를 이미 완료 → **소유효과** (내 리포트)
3. "경쟁사 매출 차트가 존재한다"는 신호를 받는 순간 → **정보 격차** 최대

### 블러 처리 방식

```
[차트 실루엣은 보이되, 숫자/내용은 안 보이게]

- CSS: backdrop-filter: blur(12px)
- 블러 위 오버레이: 반투명 흰색 + 자물쇠 아이콘
- 차트의 "형태"는 보여줘서 "가치 있는 데이터가 있다"는 인지적 자극
- 완전히 숨기는 것보다 "존재하지만 볼 수 없다"가 결제 의지를 더 강하게 촉발
```

### 페이월 카드 구성


## 4. Paid Tier — "결정적이고 치명적인" 3대 지표

Paid는 **"정보"가 아니라 "결정"**을 판다. 결제 직후 바로 실행 가능한 형태여야 한다.

### 지표 4: 업종별 매출 잠재력 (핵심)

**보여주는 것:**
- 예상 월매출 **범위** (P25 / P50 / P75) — 단일 숫자 절대 X
- "예상 월매출: **1,800만** ~ **2,400만** ~ **3,200만원**"
- 매출 분포 히스토그램 (상위 20% / 중위 / 하위 20%)
- 신뢰도 등급 (★★★★☆)

**왜 이게 치명적인가:**
- 평균 5,000만원이라는 숫자만 보면 안심하지만...
- 실제로는 상위 1개가 2억, 나머지 4개가 1,250만원인 **승자독식 시장**일 수 있다
- **"평균의 함정"**을 깨뜨려주는 분포 데이터가 핵심 결제 동기

**AI 해석 예시:**
> "이 상권의 카페 중위 매출은 2,400만원이지만, 하위 25%는 1,100만원에 불과합니다. 상위 25%에 진입해야 안정적 운영이 가능하며, 이를 위해서는 차별화된 메뉴 전략이 필수적입니다."

### 지표 5: 임대료 감당력 시뮬레이터 (BEP)

**보여주는 것:**
- 사용자가 **월세 슬라이더** 조정 → 실시간 계산
- BEP 매출 = 월 고정비 / 공헌이익률
- BEP 고객 수 = BEP 매출 / 객단가
- **"하루 커피 80잔 팔아야 손익분기점"** (직관적 표현)
- Rent-to-Sales 비율 → **안전(초록) / 주의(노랑) / 위험(빨강)** 신호등

**"보지 않으면 손해 본다" 시나리오:**
> 겉보기에 유동인구가 많은 1층 대로변 상가에 치킨집을 계약하려는데,
> 유료 데이터에서 **"해당 상권 치킨집 매출의 75%는 배달 앱에서 발생"**한다는 사실 발견.
> → 비싼 1층을 계약할 이유가 사라짐 → **매월 수백만원 임대료 절감**
> → ₩4,900으로 수천만원 손실 방지 = **압도적 ROI**

**AI 해석 예시:**
> "월세 200만원 기준, 하루 80잔(=일매출 32만원) 판매가 필요합니다. 현재 상권의 카페 중위 일매출은 28만원으로 주의 구간입니다. 월세 150만원 이하의 대안 입지를 검토하시길 권장합니다."

### 지표 6: 폐업 리스크 조기경보

**보여주는 것:**
- 향후 6~12개월 **폐업 가능성** 0~100점 게이지
- 상위 원인 Top-3 (매출 둔화 / 경쟁 급증 / 유동 감소 / 임대 압박)
- 소멸률 (영업→휴업→폐업 변환 가속도)
- **위험/주의/안전** 등급

**핵심**: "예언"이 아니라 **"조기경보"**로 포지셔닝 (법적 안전장치)

**AI 해석 예시:**
> "주변 동종 3년 폐업률이 42%로 높은 편이지만, 최근 6개월 신규 프랜차이즈 진입이 늘어 안정화 가능성이 있습니다. 핵심 리스크 요인은 ①경쟁 급증(기여도 45%) ②임대료 상승(30%) ③유동 감소(25%)입니다."

## 5. 전환율 극대화 CTA 카피

### 결제 버튼 (Sticky, 화면 하단 고정)

| 대상 | CTA 카피 | 심리 레버 |
|------|---------|---------|
| 기본 | "☕ 내 리포트 전체 보기 — ₩4,900" | 소유효과(1인칭) + 마이크로프라이싱 |
| 손실회피형 | "점포 선정 실패 비용 5,000만원 방어하기" | 손실 회피 |
| ROI형 | "₩4,900으로 수천만원 리스크 확인하기" | 구체적 ROI |
| 긴급성 | "이 상권 데이터는 다음 분기에 변경됩니다" | FOMO |

### 절대 하면 안 되는 것

- CTA 근처에 "스팸", "환불", "취소" 같은 부정적 단어 배치 → **전환율 18.7% 감소**
- 5개 이상 CTA 배치 → 단일 CTA 대비 **32% 전환율 하락**
- "Submit" 같은 무미건조한 텍스트 → **전환율 3%**에 불과

### 결제 버튼 아래 마이크로카피

```
신용카드 등록 불필요 · 카카오페이 3초 결제
이미 3,847명의 사장님이 확인했습니다
```
