## 3. 우리의 "원 터치" — 나이스비즈맵에 없는 것

### 킬러 피처: "이 자리의 역사" (개폐업 타임라인)

나이스비즈맵은 **현재 상권의 스냅샷**을 보여주지만, **"이 정확한 자리에서 이전에 뭐가 있었고, 왜 망했는지"**는 안 알려준다.

```
우리만 할 수 있는 것:

"이 주소(대전 둔산동 1234번지)에서 최근 3년간"
  2024.02 — ☠️ ○○카페 폐업 (영업 8개월)
  2023.06 — 🆕 ○○카페 개업
  2023.01 — ☠️ △△분식 폐업 (영업 1년 2개월)
  2021.11 — 🆕 △△분식 개업
  2021.05 — ☠️ □□네일 폐업 (영업 6개월)

⚠️ 이 자리에서 3곳이 연속 폐업.
   평균 생존 기간 8.7개월.
   구조적 문제(가시성/임대료)가 의심됩니다.
```

**왜 이게 킬러인가:**
1. 중개사는 절대 알려주지 않는 정보 (그들에게 불리하니까)
2. 나이스비즈맵에 없음 (카드 데이터로는 못 만드는 데이터)
3. ₩4,900의 가치를 즉시 증명 (이것 하나로 수천만원 손실 방지)
4. 우리 DB 142만건 LOCALDATA에서 바로 추출 가능

### 보조 차별화

| 차별화 | 나이스비즈맵 | 명당노트 |
|--------|----------|---------|
| **개폐업 타임라인** | ❌ 없음 | ✅ **킬러 피처** |
| **AI 종합 의견서** | 베러보스 (채팅형) | Gemini (리포트 내장형, 1,700자) |
| **가격** | 유료 (추정 월 수만원) | **₩4,900 단건** |
| **모바일 UX** | PC 중심 | **모바일 퍼스트** |
| **결론 도출** | 데이터만 제공 | **GO / 주의 / STOP 판정** |
| **공유** | PDF 다운로드 | **카카오톡 즉시 공유** |

## 8. 한 줄 전략

> **"나이스비즈맵의 뼈대를 빌리되, '이 자리에서 몇 곳이 죽었는지' 알려주는 유일한 서비스가 된다."**

나이스비즈맵이 800만건 카드 데이터로 과거 매출을 보여준다면,
우리는 142만건 점포 생사 기록으로 **"이 자리에서 살아남을 수 있는가?"**에 답한다.

그리고 답을 ₩4,900에, 모바일에서, AI 해석과 함께 제공한다.

**나이스비즈맵에 없는 것 = 우리의 무기:**
1. 📅 **개폐업 타임라인** ("이 자리에서 3곳이 연속 폐업")
2. 🚦 **GO / 주의 / STOP 판정** (데이터가 아니라 결론)
3. 🧮 **BEP 시뮬레이터** (월세 입력 → "하루 80잔")
4. 📱 **모바일 퍼스트 + 카톡 공유**
5. 💰 **₩4,900 단건** (구독 강요 안 함)
