### PAID 영역 (11개 섹션)

#### P1. 🗺 상권 지도 (경쟁 매장 시각화)
```

## 이렇게 하면 ₩4,900이 아깝지 않은 이유

```
1. 15개 섹션 — "스크롤이 끝이 안 나네?"
2. 진짜 팩트 — "142만건 실 데이터에서 뽑았대"
3. 킬러 피처 — "이 자리에서 3곳이 폐업했다고?!"
4. 인터랙티브 — "월세 조정하면 바로 계산되네"
5. AI 의견서 — "전문가한테 상담받은 느낌"

- 오픈업: 무료지만 얕음
- **명당노트**: ₩4,900에 15개 섹션 + AI + 타임라인 + 추천업종
