# 명당노트 요금제 최종 전략

> **원칙**: 나이스비즈맵 대비 약간 저렴하되, 우리만의 킬러 피처(타임라인, AI)로 가치 정당화

---

## 경쟁사 요금 비교

| 서비스 | 무료 제공 | 유료 전환 | 핵심 데이터 |
|--------|---------|---------|----------|
| **나이스비즈맵** | 가입 시 5회 풀 보고서 | 이후 건당 과금 (추정 3,000~5,000원) | 카드 실데이터 |
| **오픈업** | 12개월 매출 무료 (횟수 제한) | 프로 월 825,000원 | 카드+통신 융합 |
| **소상공인마당** | 완전 무료 | 없음 | 공공 통계 (낮은 해상도) |
| **명당노트** | **가입 시 3회 풀 보고서** | **이후 건당 ₩3,900** | 공공+AI+타임라인 |

### 포지셔닝

```
소상공인마당 (무료, 얕음)
    ↓
명당노트 (3회 무료 → ₩3,900, 깊음 + AI + 킬러피처)
    ↓
나이스비즈맵 (5회 무료 → 유료, 카드 실데이터)
    ↓
오픈업 프로 (월 82.5만원, 엔터프라이즈)
```

**우리 위치**: 소상공인마당보다 훨씬 깊고, 나이스비즈맵보다 약간 저렴하고 접근성 좋음

## 요금제 구조

### Tier 0: 비회원 (가입 전)

| 항목 | 내용 |
|------|------|
| **가격** | ₩0, 가입 불필요 |
| **제공** | Free 영역만 (종합판정 + 온도 + 피크타임 + 경쟁 요약) |
| **제한** | 일 5회 분석 |
| **목적** | "이 정도면 괜찮은데?" → 가입 유도 |

### Tier 1: 회원 무료 (가입 시)

| 항목 | 내용 |
|------|------|
| **가격** | ₩0 (카카오 로그인만) |
| **제공** | **풀 보고서 3회** (15개 섹션 전체 + AI 분석) |
| **제한** | 3회 소진 후 Tier 2로 |
| **목적** | 풀 보고서 품질 체험 → "이건 돈 내고 볼 만하다" |

**왜 3회인가 (나이스비즈맵 5회 대비):**
- 나이스비즈맵은 카드 실데이터라 5회 줘도 "정확하니까 당연"
- 우리는 공공데이터 + AI라서 "3회면 충분히 체험"
- 3회 = 후보지 3곳 비교 → 딱 의사결정에 필요한 만큼
- 너무 많이 주면 결제 동기 사라짐

### Tier 2: 단건 결제

| 항목 | 내용 |
|------|------|
| **가격** | **₩3,900/건** |
| **제공** | 풀 보고서 1회 (15개 섹션 + AI) |
| **특징** | 1건 = 1위치 × 1업종 |

**왜 ₩3,900인가:**
- 나이스비즈맵 추정 건당 3,000~5,000원 → 그 중간
- ₩4,900은 "5천원"으로 느껴짐 (좌측 자릿수 편향)
- ₩3,900은 "3천원대"로 느껴짐 → 심리적 저항 ↓
- 커피 한 잔(4,500원)보다 저렴 → "커피보다 싸네" 프레이밍

### Tier 3: 패키지

| 항목 | 내용 |
|------|------|
| **5건 패키지** | **₩14,900** (건당 ₩2,980) |
| **10건 패키지** | **₩24,900** (건당 ₩2,490) |
| **제공** | 풀 보고서 N회 (기간 제한 없음) |

**왜 패키지인가:**
- 예비 창업자: 후보지 5곳 이상 비교 시 패키지가 합리적
- 중개사: 여러 매물 분석 → 10건 패키지
- 구독 강요 없이 필요한 만큼만

### Tier 4: 월 구독 (내 가게 알림)

| 항목 | 내용 |
|------|------|
| **가격** | **₩9,900/월** |
| **대상** | 이미 매장 운영 중인 점주 |
| **제공** | 월 무제한 보고서 + 주간 알림 + AI 운영 팁 |
| **추후** | 카카오톡 푸시 알림 연동 |

### Tier 5: B2B (공인중개사/다점포)

| 항목 | 내용 |
|------|------|
| **스탠다드** | **₩49,000/월** — 무제한 + 카드뉴스 변환 |
| **프로** | **₩149,000/월** — 화이트라벨 + CRM |
